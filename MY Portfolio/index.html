AI Model Carbon Footprint Estimator: Machine Learning-Based Energy and CO₂ Prediction System

The AI Model Carbon Footprint Estimator is a machine learning-based system designed to estimate the energy consumption and carbon dioxide (CO₂) emissions generated during AI model training. The system allows users to input parameters such as hardware type, number of devices, training duration, batch size, dataset size, utilization, and data center efficiency. Using a Random Forest regression model trained on a synthetic dataset, the platform predicts the expected energy usage and corresponding carbon emissions. The project helps raise awareness about the environmental impact of AI development and supports the adoption of sustainable and energy-efficient AI practices.

AI Screenshot To HTML Code Generator

AI Screenshot to HTML Code Generator is an AI-powered tool that converts UI screenshots into structured HTML code automatically. The system analyzes the uploaded interface image using computer vision and machine learning techniques to detect layout elements such as text, buttons, images, and containers. It then generates corresponding HTML code that replicates the visual structure of the interface. This project helps developers quickly transform design screenshots into functional web layouts, reducing manual coding time and improving development efficiency.
