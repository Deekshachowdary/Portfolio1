# Personal Portfolio Website

This is a simple personal portfolio website developed to showcase my profile, technical skills, and projects. The website provides information about me and allows visitors to contact me.

## Features
- Home section with introduction
- About Me section
- Technical Skills section
- Projects showcase
- Contact form with email and LinkedIn
- Simple and responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub
- VS Code

## Sections in the Website
1. **Home** – Introduction with profile photo.
2. **About Me** – Information about my background and interest in web development.
3. **Technical Skills** – Frontend, Backend, and Tools.
4. **Projects** – Displays my developed projects.
5. **Contact** – Contact form with email and LinkedIn.

## Technical Skills

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- PHP
- MySQL
- Node.js

### Tools
- Git
- GitHub
- VS Code

## How to Run the Project
1. Open your project folder.
2. Find the file index.html.
3. Double-click index.html.
4. The website will open in your browser (like Google Chrome or Microsoft Edge).
5. Your portfolio will run.
